# Build React Native javascript code into Hermes Bytecode version 84

- Step 1:
```bash
git clone https://github.com/minhgalaxy/reactnative-build-hermes-v84.git
cd reactnative-build-hermes-v84
```

- Step 2:
```bash
npm install
```

- Step 3:
```bash
npm run build
```

- Step 4: Check `output` directory
